# Day-1 Distribution Playbook

Use this on launch day to distribute RankyPulse with channel-specific links and a direct CTA to `https://rankypulse.com/start`.

## Primary launch links

Use the pre-tagged links from:

- `docs/LAUNCH_UTM_LINKS.md`

## Required CTA format

Every post should include:

- "Try a free audit: `https://rankypulse.com/start?...UTM...`"
- One real sample report link: `https://rankypulse.com/sample-report`
